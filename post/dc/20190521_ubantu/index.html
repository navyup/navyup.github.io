[系统管理] ubuntu 16.04LTS平滑升级至18.04

升级背景

近日，计划使用AWS主机安装jupyter时发现，主机操作系统版本是ubuntu 16.04LTS，无论如何更新Python3只能停留在3.5版本，然后决定升级操作系统。 因AWS主机上还跑有其他应用，因此选择升级平滑升级至ubuntu 18.04

升级步骤

第一步，更新系统

使用命令先更新现有系统至最新版本

1
2
3

$ sudo apt-get update
$ sudo apt-get  -y upgrade
$ sudo apt-get -y dist-upgrade

第二步，升级

开始升级，期间需要重启操作系统，下载很多软件，并询问你多次，直接一Y到底即可。

1

$ sudo do-release-upgrade

第三步，确认python3环境

1

$python3 -V
